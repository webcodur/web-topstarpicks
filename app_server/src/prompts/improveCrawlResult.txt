이는 유명인사의 인터뷰를 바탕으로 그들의 추천 책 데이터가 주어집니다.

각 책에 대해 속성을 수정, 추가해 주세요.

정보를 찾을 수 없거나 환각(hallucination)이 의심되는 경우, 해당 필드를 비워두세요 (빈 문자열 사용).

입력: 책 정보
{
    "title": "영문 책 이름",
    "creator": "영문 책 저자",
    "img_link": "이미지 URL",
    "recommendation_text": "추천 내용",
    "recommendation_source": "추천 출처 링크"
},

출력: 개선된 책 정보의 JSON 파일
{
  "title": "영문 책 이름 그대로",
  "creator": "한국어로 저자 이름 번역",
  "img_link": "이미지 URL",
  "recommendation_text": "번역된 추천사 텍스트 (양식 정보 및 추천자 인물명 제거, 부드러운 말투 및 경어체)",
  "recommendation_source": "추천 출처 링크",
  "release_date": "출시일 (YYYY-MM-DD 형식, 알 수 없는 경우 빈 문자열)",
  "mediaDescription": "추천인과 상관없는 책에 대한 객관적인 설명 (4줄 분량의 한국어 설명, 위키피디와와 같은 권위적인 말투. 경어체 사용하지 않음)"
},

만약 주어진 인물에 대한 정보가 불충분하거나 평가가 불가능한 경우, 다음과 같이 응답하세요:
{
  "error": "평가 불가: [구체적인 이유]"
}
