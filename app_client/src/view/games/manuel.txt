## 1. 게임 기본 구성

### 1.1 게임 개요

- 게임 방식: 턴제 방식의 1:1 대전 게임
- 승리 조건: 상대방 체력을 0으로 만들기
- 시작 체력: 100
- 매 턴마다 하나의 주제가 주어지며, 다음 턴의 주제도 미리 공개됩니다.

### 1.2 카드 시스템

- 전체 카드 수: 30장
- 시작 시 각 플레이어는 4장의 카드를 받습니다
- 매 턴 1장의 카드를 사용해야 합니다
- 매 턴 1장의 카드를 자동으로 보충받습니다

### 1.3 카드 정보

- 구성 요소: [이름, 직군, 등급]
- 등급별 기본 점수:
    - S등급: 60점
    - A등급: 50점
    - B등급: 40점
    - C등급: 30점
    - D등급: 20점

### 1.4 직군별 특수 효과

- 지도자: 모든 행동의 기본 점수 +15
- 정치인: 외교 행동 선택 시 +25
- 지휘관: 교전 행동 선택 시 +30
- 기업가: 통치 행동 선택 시 +25
- 투자자: 상대방의 점수를 15 감소
- 학자: 모략 행동 선택 시 +25
- 예술인: 현재 주제와 같은 행동 선택 시 추가로 +15
- 작가: 모든 행동의 기본 점수 +10, 주제 보너스 +5
- 배우: 상대 카드의 직군 점수 보너스를 복사
- 인플루엔서: 주제 보너스를 2배로 적용
- 스포츠인: 체력이 40 이하일 때 모든 점수 +20

## 2. 게임 진행

### 2.1 턴 구조

1. 턴 시작
    - 현재 턴 주제 확인
    - 다음 턴 주제 공개
2. 드로우 단계
    - 덱에서 카드 1장 드로우
3. 행동 단계
    - 4가지 행동 중 1개 선택
    - 카드 1장 사용 필수

### 2.2 주제 시스템

- 주제 종류: [통치, 외교, 교전, 모략]
- 주제 보너스: 현재 턴 주제와 같은 행동 선택 시 +20점

## 3. 행동 시스템

### 3.1 행동 종류 및 효과

1. **통치**
    - 효과: 자국 체력 회복 30
2. **외교**
    - 효과: 자국 체력 +15, 상대 체력 -15
3. **교전**
    - 효과: 상대 체력 -30
4. **모략**
    - 효과: 승리 시 상대방 행동의 효과를 대신 획득
    - 모략 vs 모략: 패자에게 15 피해

## 4. 승부 판정

### 4.1 기본 규칙

- 점수 계산: [등급 점수 + 직군 특수 효과 + 주제 보너스]
- 높은 수치가 승리하여 효과 발동
- 패자의 효과는 발동되지 않음
- 완전 동점: 무승부로 처리 (양측 효과 미발동)

## 5. 게임 종료

### 5.1 종료 조건

- 한 진영 체력 0 이하 도달 시
- 체력이 높은 플레이어 승리
